# olio-ohjelmointi-harjoitustyo
Simple java desktop application with java backend

## Architecture
- Postgress 18 db
- Desktop app Java
    - Javafx + atlantafx
- Backend Java
    - Quarkus providing rest api

## Running

Clone git repo
```bash
git clone https://github.com/Mikxus/olio-ohjelmointi-harjoitustyo.git
```


To run locally
```
./gradlew run
```
